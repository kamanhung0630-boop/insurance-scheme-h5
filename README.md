# 友邦保险教育金方案 H5 页面

## 项目概述
这是一个为香港保险经纪设计的储蓄保险方案展示H5页面，专为35岁女性客户量身定制的HKD教育金储蓄方案。

## 功能特点
- 响应式设计，优化移动端体验
- 交互式保费计算器
- 现金价值增长图表
- 繁体中文支持
- 预约咨询表单
- 分享功能
- 导出PDF功能
- 滚动动画效果
- PWA支持

## 使用说明
1. 页面加载后自动转换为繁体中文
2. 调整保费滑块查看不同方案效果
3. 点击底部悬浮按钮进行预约、分享或导出PDF
4. 导出PDF使用浏览器打印功能（选择"保存为PDF"）
